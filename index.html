Scoring pattern
Discovery-skip is now a first-class deficiency
The base prompt penalized agents for outcomes — wrong referral, no sign, dropped call. SML rules penalize agents for process failures that lead to those outcomes. Skipping the required discovery questions before declining is now a 10–15pt deduction on its own, regardless of whether the case was actually non-viable.
Grading pattern
Correct partial positives now get credit
The base prompt is binary on many coaching points. SML's no-penalty scenarios introduced nuance: an agent who gives an accurate SOL warning (even if incomplete) gets partial credit. An agent who re-engages a caller mid-call after an initial misstep gets partial credit. Two agents scored higher after re-analysis because of this.
Rescue pattern
Escalation failure is now distinct from conversion failure
The base prompt conflates "didn't sign" with "rescue-worthy." SML separates them. A case where the agent correctly scheduled a callback is not a rescue under Section 4. A case where the agent missed an Auto-Escalate trigger is always a rescue, even if the agent was otherwise competent. These are now evaluated independently.
Coaching quality shift
From generic intake advice to firm-specific, actionable instruction
The most significant qualitative change isn't in scores or grades — it's in what agents are actually told to do differently.
1
Section references replaced vague guidance. "Agent should gather more information before declining" became "Agent must apply SML's Section 2 discovery questions — specifically compensable harm, provider identity, and negligence type — before any declination." Agents now know exactly what protocol was violated.
2
Escalation coaching is now trigger-specific. "Agent should have escalated sooner" became "Post-operative sepsis following elective knee replacement qualifies as an SML Red Flag Injury under Section 2 and requires immediate routing to an attorney or medical reviewer, not a records-first hold." The trigger is named; the required action is named.
3
False negatives are now caught in coaching too. Kenneth Bucknam's coaching in the base prompt was single-focus (language neutrality). SML rules added a second coaching track: isolate physically verifiable claims from delusional ones using SML's discovery sequence, so the legal team receives a structured triage rather than an undifferentiated narrative.
4
Correct behavior is now explicitly credited. Agents who followed SML protocol correctly (escalation transfers, informed consent declines with no harm) receive affirmative coaching notes — not silence. The base prompt tends to note deficiencies only; SML instructions add a reinforcement layer for correct decisions.
